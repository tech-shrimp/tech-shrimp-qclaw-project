### WeRSS项目部署

https://github.com/rachelos/we-mp-rss

快速开始，找一台能运行docker的服务器或电脑
```
docker run -d  --name we-mp-rss  -p 8001:8001 -v ./data:/app/data  ghcr.io/rachelos/we-mp-rss:latest
```

1. 启动服务后，访问 http://<您的IP>:8001 进入管理界面。
2. 注册一个自己的微信公众号
3. 使用微信扫码授权后（用公众号角色授权），即可添加和管理订阅。
4. 软件会自动追踪内容，并生成RSS订阅链接。

### 开发学习路线图

提示词：
```
创建一个「技术学习路线图 Skill」。  
  
用户会输入一个想学习的主题和目标，你的任务是：结合用户基础和Exa搜索结果，生成一条可执行的学习路线图。 以下是Exa SDK的文档："C:\Users\techshrimp\.qclaw\workspace-agent-829bf46e\exa-python-cli-dev-doc_20260702.md" 
  
工作流程：  
  
1. 先判断用户水平，如果用户只给了学习主题，需要先反问：  
- 你现在是什么水平？有没有编程基础？  
- 你想达到什么目标？入门、做项目、就业、作品集还是商业化？  
- 每周准备投入多少时间，预计学习多少周？  

2. 调用Exa搜索能力  根据学习主题搜索相关资料，并综合分析：标题，简介，内容 - 如果用户基础不足，先安排补基础阶段。
- 进行多轮搜索，覆盖多种难度，覆盖实战与基础概念讲解
- 对每一个阶段学习内容都进行搜索
 
3. 生成分阶段路线图， 路线图需要包含n个阶段，每个阶段包括：  
- 阶段名称  
- 学习目标  
- 推荐教程
- 知识作业（选择题+问答题，不用附答案）
- 实战作业
  
4. 最终需要输出：    
- 分阶段学习路线
- 为每个阶段新建一个项目文件夹
- 每个文件夹包含学习指南，知识作业，实战作业，文件名用中文。
- Mermaid 格式路线图

要求：  
- 不要凭空编造，必须基于搜索结果推荐。    
- 每个阶段都要有实际作业。  
- 输出要清晰、具体、可执行。
```


设置EXA环境API KEY：
- `EXA_API_KEY` 获取地址 ：!(https://dashboard.exa.ai/api-keys)[https://dashboard.exa.ai/api-keys]
- API Key设置方法:
  - Windows: setx EXA_API_KEY "your_secret_here"
  - MacOS: 在 `~/.zshrc` 末尾追加 `export EXA_API_KEY='your_secret_here'`
- API Key 设置完成以后，需要重启QClaw软件.
